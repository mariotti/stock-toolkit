# Motivations

One question: how can I make money with AI?

Ask AI: How can I make Money?

If there was a single answer we will all make the same thing,
and eventually stop to make money.

In my opinion the obvious simple test is the stock market:
everybody can ask: Dear AI what should I buy/sell today?

# A different approach to money

The first 2 points were to be able to collect information
and to use that information in classical stock market parameters:

 - How can I collect stock data?
 - What are the standard stock market parameters?

So the question changed: Given that I have these data,
how can I make money?

The problem than changes based on which data we have.
